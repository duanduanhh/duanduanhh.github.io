OpenAI API绑卡充值教程

发表于2024-02-06|更新于2025-02-05
|阅读量:

前言：OpenAI的用户分为网页用户与开发者。两类用户之间独立收费，互不相通。开通GPT Plus的用户，每月收费标准为20美元，购买后可以在网页入口选择GPT4模型进行聊天交互。开发者用户获取OpenAI API Key的使用权限（包括3.5模型）需要充值费用，每个账户默认有5美刀的基础费用，3个月时效期。保证Key对应的账户内有充足的余额才可正常调用，不同的token数量进行有不同的收费标准。所以开发者用户想要调用API，如果你的账户内没有足够余额，需要自己先开通一张虚拟信用卡，后续进行绑卡充值才可调用。如果你没有虚拟信用卡，可以可以点击右侧链接进行注册：https://yeka.ai/i/GPT5


绑定教程：

  1. 点击进入OpenAI官网，登录你的个人账户，选择Settings - Billing - Add payment details

  1. 选择Individual

  1. 将你开通的信用卡信息对标着填入框内

  1. 绑卡结束后会让你选择充值金额，可以是5~50美元之间的任意数额，充值后即可使用

  1. 点击API Keys-Create new secret key创建你的个人Key，注意保存

然后就可以在代码里使用啦！
