二、实现调用API脚本


常见错误：

  1. Q：openai.error.RateLimitError: You exceeded your current quota, please check your plan and billing details.

    A：账户余额不足，调用API实行收费机制，每个账号默认有5刀，但只有三个月有效期，过了三个月或者5刀用完需要单独充值


体验：
